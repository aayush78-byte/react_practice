# React Password Generator

A simple and responsive password generator built using React and Tailwind CSS.

## Features

- Generate random passwords
- Customize password length
- Include numbers option
- Include special characters option
- Dynamic password generation using React Hooks
- Responsive UI

## Tech Stack

- React.js
- Tailwind CSS
- JavaScript
- Vite

## Concepts Used

- useState()
- useEffect()
- useCallback()
- Controlled Components
- Event Handling
- Random Password Generation Logic

## Installation

Clone the repository

git clone <your-repository-link>

Install dependencies

npm install

Run project

npm run dev
